# 🛠️ Welcome to CodeCurrent Platform Tooling

> **Shared tooling, templates, and developer-experience resources for the CodeCurrent enterprise**

---

## 🎯 About This Organization

**CodeCurrent Platform Tooling** provides reusable tooling and templates that help teams ship software faster and more consistently. Everything here is designed to be picked up and used across projects with minimal setup.

| Category | Examples |
|----------|---------|
| 🚀 **Codespaces Templates** | Pre-configured dev containers for quick onboarding and consistent environments |
| 📋 **Repository Templates** | Starter repos with CI/CD, linting, and project structure baked in |
| ⚙️ **Workflow & Action Libraries** | Shared GitHub Actions workflows for builds, tests, releases, and security checks |
| 📦 **Automation Tooling** | Scripts and utilities for common delivery tasks |

## 🚀 What You'll Find Here

- **GitHub Codespaces Templates** — Ready-to-use dev container configurations so every contributor gets a working environment in seconds
- **CI/CD Starter Kits** — Opinionated GitHub Actions workflows for build, test, and deploy pipelines
- **Repository Templates** — Scaffolded repos with best-practice structure, tooling, and documentation
- **Quality & Security Tooling** — Shared configurations for linting, scanning, and code review automation
